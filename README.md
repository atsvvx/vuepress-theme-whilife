
# vuepress-theme-whilife

一个基于时间线的 vuepress 主题的博客 

## 已经完成

- 博客功能
基于 `@vuepress/plugin-blog` 插件提供的功能, 已经完成了初步的博客功能. 现在支持包括 Post(博文) 以及 frontmatters 实现的 Tag.
